
# Dakdak Management

Polymer 1.0 -based upload & metadata management app for "Dakdak image gallery". Targeted to serve skateboarding pictures.

Uses the [Dakdak API](https://github.com/miro/dakdak-api)

*Strongly in Work-In-Progress mode!*
